About My Media Player:
DEPENDENCIES:
	FFMPEG.exe
	FFPLAY.exe
	FFPROBE.exe
	SoundVolumeView.exe

	The FFMPEG Executables Are Best Downloaded Here. https://github.com/yt-dlp/FFmpeg-Builds
	SoundVolumeView Download Can Be Found Here. https://www.nirsoft.net/x64	_download_package.html


My Media Player Plays All FFMPEG File Extensions Including Audio And Video. Image Files Are Played
Using CV2. The Volume Control Utilizes PYCAW For Audio Control And The Stream Is Directed To
Selected Devices Using SoundVolumeView. Also Included Is A YouTube Video And Audio Downloader.
