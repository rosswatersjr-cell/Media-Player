About My Media Player:
DEPENDENCIES:
	FFMPEG.exe
	FFPLAY.exe
	FFPROBE.exe
	SoundVolumeView.exe

	The FFMPEG Executables Are Best Downloaded From https://github.com/yt-dlp/FFmpeg-Builds
	Since There Are Fixes For YT-DLP Which Is Used For The Downloading. Just Select Your Windows
	System Type And Download. SoundVolumeView Download Can Be Found At https://www.nirsoft.net/x64	_download_package.html


My Media Player Plays All FFMPEG File Extensions Including Audio And Video. Image Files Are Played
Using CV2. The Volume Control Utilizes PYCAW For Audio Control And The Stream Is Directed To
Selected Devices Using SoundVolumeView. Also Included Is A YouTube Video And Audio Downloader.
