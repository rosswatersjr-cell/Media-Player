# Media-Player
Created Using Python 3.14.2. 
Media Player Program For Playing Audio, Video And Image Files For Windows 11. 
DEPENDENCIES: FFMPEG.exe, FFPLAY.exe, FFPROBE.exe And SoundVolumeView.exe.
These 3 ffmpeg Executables are Installed Automatically The First Time The Program Executes.
SoundVolumeView Download Can Be Found Here. https://soundvolumeview.en.softonic.com/ 
Place SoundVolumeView Executables In Your Script Folder.
