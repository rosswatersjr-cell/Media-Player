Acerca de Mi Reproductor Multimedia: 

DEPENDENCIAS: 
	FFMPEG.exe 
	FFPLAY.exe 
	FFPROBE.exe 
	SoundVolumeView.exe 

Los ejecutables de FFMPEG son mejor descargarlos desde https://github.com/yt-dlp/FFmpeg-Builds Ya que hay correcciones para YT-DLP que se utiliza para la descarga. Solo selecciona tu tipo de sistema Windows y descarga. La descarga de SoundVolumeView se puede encontrar en https://www.nirsoft.net/x64_download_package.html Mi Reproductor Multimedia reproduce todas las extensiones de archivos FFMPEG, incluyendo audio y video. Los archivos de imagen se reproducen utilizando CV2. El control de volumen utiliza PYCAW para el control de audio y la transmisión se dirige a dispositivos seleccionados utilizando SoundVolumeView. También se incluye un descargador de videos y audios de YouTube.
